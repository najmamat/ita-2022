# [matous.najman.cz](matous.najman.cz) Portfolio Site

Clone the repository to get started.

## [](#clone-repository)Clone

```
git clone https://github.com/najmamat/portfolio.git
```

### [](#install)Install

```
npm install
```

### [](#start)Start

```
npm run start
```
